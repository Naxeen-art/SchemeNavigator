# Scheme Navigator AI

An intelligent system to help citizens find relevant government schemes using AI-powered search.

## Features

### User Portal
- 🔍 Smart scheme search using AI
- 📊 Real-time scheme matching
- 📝 Detailed scheme information
- 🚀 Quick actions for documents and application
- 📜 Query history tracking

### Admin Panel
- 📊 Dashboard with analytics
- 📁 Multiple data upload formats (JSON, CSV, Form)
- 🔍 User query monitoring
- ⚙️ Scheme management
- 📥 Data export capabilities
